Contact Management System: Feature Summary

Feature and description:

Core CRUD -	Full ability to Create, View, Update, and Delete contacts.
Data Security -	Duplicate Check Prevents saving the same phone number twice.
Persistence	- Uses SQLite to ensure all data is saved even after the app is closed.
Search & Filter -	Case-insensitive search by name with a "Show All" reset button.
Smart View - Toggle List: Hide or show the contact list with a single click to keep the UI clean.

4 Additional Premium Features:

1. Favorite Contacts (Starring)
- Mark important contacts with a gold star.
- Easily toggle favorite status directly from the list.

2. Advanced Sorting
- A dedicated toolbar button to switch between Alphabetical (A-Z) and Recent (Newest first) views.
- The icon changes dynamically to show your current sorting mode.

3. Live Contact Count
- A persistent badge at the bottom of the screen showing your Total Contact Count.
- Updates instantly whenever a contact is added or removed.

4. Instant Dark Mode
- A toggle switch in the header to alternate between Light Mode and a sleek Dark Blue Mode.
- Perfect for saving battery and reducing eye strain.
